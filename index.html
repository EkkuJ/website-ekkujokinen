Get-to-Class!

A dashboard that lists the upcoming public transport from my home to Aalto University. It uses the digitransit GraphQL API. The application is also Dockerized and the container image can be found from this repository.

Have a look GitHub Repo

Key words: React, GraphQL, Docker

Game X Changers

A game library built with Python where users can buy and play games. Developers can add their own games and get sales statistics trough the application or an API. Check GitHub for credentials or authenticate via Facebook!

Have a look GitHub Repo

Key words: Django, OAuth, API

Restaurant Sort

This is the first web application by me. It sorts restaurats provided by WOLT in an efficient but not-that-pretty manner. It's shown here mainly to demonstrate the progress I've made (compare to this website, for example).

Have a look GitHub Repo
